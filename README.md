# EFT · Quest Planner

Интерактивный планировщик квестов Escape from Tarkov: граф зависимостей всех
квестов, цели и предметы, оптимальный путь по экспе, планировщик рейдов —
и прогресс до контейнера Каппа в том числе.

## Возможности

- **Граф** — все квесты по торговцам с линиями зависимостей, зум/панорама (мышь и тач),
  подсветка цепочки пререквизитов по клику.
- **Список** — табличный вид с сортировкой по торговцам.
- **Рейды** — планировщик: какие квесты закрывать на каждой карте.
- Режимы **1.0.6.5 (старый)** / **1.1.0.0 (новый)** — разные требования Каппы.
- Отметка «сделано» с автоотметкой всех пререквизитов; фильтры по типу, состоянию,
  торговцу, мин. XP; учёт уровня персонажа; сезонный бонус +25% XP.
- RU / EN интерфейс, 4 цветовые темы.
- Прогресс хранится в localStorage; опционально — облачная синхронизация через
  Supabase (email/пароль или Google), с объединением локального и облачного прогресса.

## Структура

| Файл         | Назначение                                          |
| ------------ | --------------------------------------------------- |
| `index.html` | Разметка + загрузчик данных                         |
| `styles.css` | Все стили (включая мобильную версию)                |
| `app.js`     | Логика приложения                                   |
| `tasks.json` | Данные квестов (единственный источник, 510 квестов) |

Сборки нет — статические файлы. Хостится на Vercel: деплой — просто пуш в `main`.

## Локальный запуск

Данные загружаются через `fetch`, поэтому нужен любой статический сервер
(с `file://` не заработает):

```bash
npx http-server . -p 8080
# затем открыть http://localhost:8080
```

## Тесты

Смоук-тест на Playwright (граф, отметка квестов, фильтры, мобильная вёрстка):

```bash
npm install
npx playwright install chromium
npm test
```

Тот же тест гоняется в GitHub Actions на каждый пуш.

## Обновление данных квестов

`tasks.json` — единственный источник данных. `id` квестов совместимы с
[tarkov.dev API](https://tarkov.dev/api/).

Цели квестов (таргеты) и вики-ссылки обновляются скриптом:

```bash
node scripts/update-objectives.mjs
```

Скрипт тянет объективы (EN + RU) и `wikiLink` из открытого GraphQL API
tarkov.dev и дополняет `tasks.json`, не трогая остальные поля. Сейчас цели
заполнены примерно у 210 квестов (все каппа-квесты); прогон скрипта с машины
с открытым доступом к `api.tarkov.dev` добьёт остальные и добавит русские
тексты целей.
